git分布式版本控制系统

我第一个接触的版本控制系统是svn，当时觉得版本控制就是这样，直到我遇到了git…
git是分布式版本控制系统，合适分布式开发，强调个体。速度快、灵活，代码冲突了也比较好解决，最让我喜欢的还是git的分支切换。
